Getting Started

This simulator has been designed to give a detailed and lucid explanation of the MIPS processor. For the sake of conceptual understanding, each instruction has been divided into 7 micro-instructions. The simulator provides its users an option to create their own instructions using the same architecture thereby providing detailed understanding of the subject.This guide provides a brief tutorial on using this simulator.

INSTRUCTION BOX

In the instruction box upto 10 instructions can be entered at once. WIth each instruction its corresponding address i.e its pc value is displayed. The instructions can be loaded into the instruction memory by clicking the "load instruction button".
The reg file can be viewed and modified by pressing the "reg file button". The "refresh button" clears the instruction box.
