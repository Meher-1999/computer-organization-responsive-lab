Single Cycle MIPS CPU

Colour Conventions

Gray coloured arraws signify previous cycle's values/ values not modified by the current instruction till now.


Active green coloured arrows signify the values being modified in the current micro-step of the current instruction.


Ligt green coloured arrows signify the values which have been modified by the current instruction but in the previous micro-steps.


Red colored arrows signify the values which were processed but were not selected to flow in the later part of execution.
